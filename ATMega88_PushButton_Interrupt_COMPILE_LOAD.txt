The compilation and object code processing commands:

1. avr-gcc -g -Os -mmcu=atmega88 -c ATmega88_PushButton_Interrupt.c 

2. avr-gcc -g -mmcu=atmega88 -o ATmega88_PushButton_Interrupt.elf 
ATmega88_PushButton_Interrupt.o

3. avr-objcopy -j .text -j .data -O ihex ATmega88_PushButton_Interrupt.elf ATmega88_PushButton_Interrupt.hex

To Program the AVR:
avrdude -c avrispmkII -p m88 -U flash:w:ATmega88_PushButton_Interrupt.hex
